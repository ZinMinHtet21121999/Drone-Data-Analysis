# 🚁 Drone Flight Performance Analysis

## About
Data Analyst | Python | SQL | Drone Operation

## Project Overview
Analyzes drone flight performance using 
battery capacity, weight and speed metrics.

## Tools
- Python (Pandas, Numpy)
- Matplotlib & Seaborn
- SQL (SQLite)

## Key Insights
- Higher battery capacity = longer flight
- Heavier drones drain battery faster
- Battery efficiency is the main performance factor
